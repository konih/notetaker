## Roadmap

### Phase 1 (current)

- CLI
- audio capture (PipeWire/PulseAudio monitor)
- chunked OpenAI transcription
- SQLite persistence
- Markdown export

### Phase 2

- speaker diarization (pluggable provider)
- better realtime streaming / incremental transcription
- session search and filtering
- improved summaries (templates, metadata)

### Phase 3

- web UI or Tauri UI
- live transcript view
- annotations and highlights
- meeting templates
- Jira/Confluence export
